# Проект виджета банковских операций 
## Цель проекта: написать виджет, позволяющий обеспечить удобное и быстрое взаимодействие клиента с системой банка


## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/username/project-x.git
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
3. Вызовите файл main.py:
```
python main.py
```

Функционал программы написан в пакете src, каждый модуль отвечает за определенную часть программы, в каждой функции присутствует docstring
